LIB2024- QA

  1. Qus: What are the three components of Web?

    Ans: The three components of the Web are HTML, CSS, JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. An example of version control is GitHub.

  3. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The <p> tag is used to create a paragraph in HTML.

    Example: <p>This is a paragraph.</p>

  4. Qu: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: The <p> creates paragraphs. The <a> defines a hyperlink. The <i> defines a part of text in an alternate voice or mood usually in italics. I think the odd one is the <span> because it is used to group inline-elements in a document.

  5. Qu: What are the tags required to create a navigation list in HTML?

    Ans: The tags; <nav>, <ul> and <li> are required to make a navigation list in HTML.

  6. Qu: What element is used to create images? Name its attributes.

    An: The <img> attribute is used to create images. <src>, <alt>, <height> and <width> are attributes of the img element.

  7. Qu: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    An: The <table> tag is used to create a table in HTML. The <tr> tag defines a row in an HTML table. The <td> tag defines a standard cell in an HTML table. The odd one is the <b> tag as it creates bold text in HTML because it is not found neccessary in a <table> tag.
