  11. Qus: Give an example of inline style.

    Ans:
    Inline style is found inside the individual tag of an HTML document so that it may override the the value of the font-style css property. An example of inline style within a section tag would look like this.

    <section style="font-style:italic;">
    This will make the font within the section italic.

  12. Qus: How would you link an external style sheet? Give example.

    Ans:
    You can link an external style sheet by using the link tag in the head of your HTML.
    The link tag looks like this.
    <link rel="stylesheet" type="text/css" href="css/style.css">
    The href defines where the file is located. css/ is the folder the file style.css is found in.
